# WM-09. 世界模型评测：物理推理基准

## 1. 评测的根本转变

传统视频生成模型的评测标准是"好不好看"：FID、FVD 等指标衡量**视觉质量**。

但世界模型的核心能力不是"画得好看"，而是"懂不懂物理"：

```
传统评测:  "生成的视频分辨率高不高？色彩自然不自然？" → FID/FVD
世界模型评测: "生成的视频中，物体是否遵守物理定律？" → 物理一致性
```

> **类比**：评价一个建筑设计师，不是看效果图画得漂不漂亮，而是看设计是否结构合理、不会倒塌。

## 2. 四大评测基准

### (a) WorldModelBench（2026.02）

**最新最全面**的世界模型评测基准。

**核心创新**：检测视频中**细微的物理违规**：
*   物体大小异常变化（违反质量守恒）
*   物体穿模（违反不可穿透性）
*   运动轨迹不自然（违反惯性定律）
*   因果关系错误（原因和结果不匹配）

| 指标 | 数值 |
| :--- | :--- |
| 人工标注样本 | **67,000** 个 |
| 评测模型数 | 14 个前沿模型 |
| 比 GPT-4o 检测准确率高 | +8.6% |

**评测维度**：
1.  物体持久性（不凭空消失/出现）
2.  质量守恒（大小不异常变化）
3.  运动一致性（不违反牛顿定律）
4.  交互合理性（碰撞、遮挡正确）

### (b) PhysBench（2025.01）

评测 VLM 对**物理世界的理解能力**。

| 指标 | 数值 |
| :--- | :--- |
| 数据量 | 10,002 个视频-图像-文本条目 |
| 评测模型数 | **75 个** VLM |
| 领域 | 4 个（物体属性、关系、场景、动力学） |

**关键发现**：
*   模型在**常识推理**上表现尚可。
*   在**物理现象理解**上普遍很差。
*   原因：训练数据中缺少物理知识。

### (c) PhysReason（2025.02）

评测 LLM/VLM 的**物理推理能力**（需要多步计算）。

| 指标 | 数值 |
| :--- | :--- |
| 问题数量 | 1,200 个 |
| 平均解题步骤 | 8.1 步 |
| 最难题目步骤 | 15.6 步 |

**结果令人震惊**：
*   Deepseek-R1（最强模型）：< 60% 准确率。
*   知识类问题：75.1%。
*   困难推理问题：**31.95%** ← 远未达到人类水平。

### (d) SmallWorld

在**受控孤立环境**中评测世界模型的动力学理解：

| 领域 | 评测内容 |
| :--- | :--- |
| 刚体碰撞 | 两个球碰撞后的轨迹 |
| 流体 | 水流的运动模式 |
| 弹性 | 弹性物体的变形和回弹 |
| 摩擦 | 不同表面的滑动行为 |
| 重力 | 自由落体和抛物运动 |
| 布料 | 织物的悬挂和褶皱 |

**评测对象**：Transformer、扩散模型、Neural ODE 等不同架构。

## 3. 传统 vs 物理评测指标

| 指标 | 评测什么 | 适用于世界模型？ |
| :--- | :--- | :--- |
| **FID** | 图像质量/多样性 | ❌ 高 FID 不代表物理正确 |
| **FVD** | 视频质量 | ❌ 同上 |
| **SSIM/PSNR** | 像素级相似度 | ❌ 物理正确但视觉不同时会低分 |
| **CLIP Score** | 图文匹配度 | ❌ 不评测物理 |
| **物理一致性** | 是否遵守物理定律 | **✅ 核心指标** |
| **因果正确性** | 因果关系是否正确 | **✅ 核心指标** |
| **长期一致性** | 长视频中是否保持一致 | **✅ 核心指标** |
| **可交互性** | 能否响应动作输入 | **✅ 核心指标** |

## 4. 当前模型的物理理解水平

根据各基准的评测结果：

| 物理能力 | 当前水平 | 人类水平 |
| :--- | :--- | :--- |
| **物体持久性** | 中等（短期可以，长期不行） | 完美 |
| **重力** | 基本正确 | 完美 |
| **碰撞** | 简单碰撞可以，复杂碰撞不行 | 完美 |
| **流体** | 很差 | 直觉即可 |
| **布料/绳索** | 很差 | 直觉即可 |
| **因果推理** | 差（31.95% 在困难题目上） | ~95% |
| **反事实推理** | 极差 | 容易 |

## 5. 为什么物理理解如此困难？

| 原因 | 说明 |
| :--- | :--- |
| **训练数据偏差** | 互联网视频以"正常"场景为主，缺少"异常"物理现象 |
| **隐式学习** | 模型从像素中隐式学习物理，没有显式的物理方程 |
| **长尾分布** | 极端物理现象（爆炸、碎裂）数据极少 |
| **接触动力学** | 碰撞、摩擦等接触物理是最复杂的物理模拟问题 |
| **多尺度** | 宏观运动和微观变形需要不同尺度的建模 |

## 6. 改进方向

| 方向 | 说明 | 代表 |
| :--- | :--- | :--- |
| **物理先验注入** | 在模型中加入显式的物理约束 | Physics-informed Neural Networks |
| **仿真数据增强** | 用物理引擎生成精确的物理场景作为训练数据 | Isaac Sim + Cosmos |
| **多步推理** | 让模型学会分步骤推理物理过程 | Chain-of-Thought for Physics |
| **主动实验** | 让模型在仿真中"做实验"验证物理假说 | World-VLA-Loop |

> **一句话总结**：世界模型评测正从"好不好看"转向"懂不懂物理"——WorldModelBench 等基准首次系统化评测物理一致性，揭示了当前模型的物理理解仍远低于人类水平（困难推理题仅 32%）。"让 AI 真正理解物理世界"仍是世界模型领域最大的开放挑战。
