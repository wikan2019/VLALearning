# WM-01. 世界模型概述：AI 的想象力引擎

## 1. 什么是世界模型？

**世界模型 (World Model)** 是 AI 系统内部对外部世界的**可模拟表示**。它能够：
1.  **理解当前状态**：知道世界现在"是什么样"。
2.  **预测未来状态**：给定一个动作，预测世界会"变成什么样"。

```
                    ┌─────────────────┐
当前观测 ──→       │   🧠 世界模型     │  ──→ 预测的未来状态
动作/指令 ──→      │  (内部模拟器)     │  ──→ 预测的奖励/结果
                    └─────────────────┘
```

> **一句话定义**：世界模型 = AI 的想象力。它让 AI 不用真正尝试就能预测行动的后果——"如果我推这个杯子，它会掉下去吗？"

## 2. 人类的世界模型

人类天生拥有世界模型——这就是**直觉物理 (Intuitive Physics)**：

*   你不需要真的把杯子推到桌边，就知道它会掉下去。
*   你不需要真的开车上高速，就知道急转弯会翻车。
*   你不需要真的把冰放到火上，就知道它会融化。

这些"脑中预演"正是世界模型的本质——**在内部模拟中推演因果关系**。

## 3. 世界模型在 AI 技术栈中的位置

```
┌─────────────────────────────────────────────────────┐
│                     AI 技术栈                        │
│                                                     │
│  LLM   →  理解和生成文字（知识推理）                  │
│  VLM   →  理解图像和视频（视觉感知）                  │
│  VLA   →  输出机器人动作（物理执行）                  │
│  World Model → 🌍 预测世界变化（内部模拟）  ← 本篇    │
│                                                     │
│  关系：World Model 为 VLA 提供"想象中的练习场"       │
│        VLM 为 World Model 提供"感知输入"             │
│        LLM 为 World Model 提供"语义推理"             │
└─────────────────────────────────────────────────────┘
```

## 4. 世界模型的两大功能

### (a) 理解当前世界 (World Understanding)

```
输入: [一帧图片：球在斜坡顶部]
世界模型理解: "这是一个球，在斜坡上，受重力影响，即将滚下"
```

这需要模型具备：
*   **物体识别**：知道什么是球、什么是斜坡
*   **物理知识**：知道重力、摩擦、碰撞等规律
*   **空间理解**：知道物体的相对位置和几何关系

### (b) 预测未来状态 (Future Prediction)

```
输入: [当前帧：球在斜坡顶部] + [动作：轻推一下]
世界模型预测: [未来帧序列：球沿斜坡滚下 → 到达底部 → 撞到墙壁 → 弹回]
```

这需要模型具备：
*   **动力学建模**：预测物体运动轨迹
*   **因果推理**：理解"推"导致"滚"
*   **长时预测**：不只预测下一帧，而是预测未来一系列帧

## 5. 世界模型的四大表示形式

| 表示形式 | 预测什么 | 优点 | 缺点 | 代表 |
| :--- | :--- | :--- | :--- | :--- |
| **像素级视频** | 未来的 RGB 帧 | 直观、可视化 | 计算量巨大、容易模糊 | Sora, Genie |
| **潜在空间 (Latent)** | 压缩特征空间中的状态 | 高效、信息密度高 | 不直观、需要解码器 | Dreamer, V-JEPA |
| **占用网格 (Occupancy)** | 3D 体素空间的占用概率 | 几何精确 | 分辨率受限 | OccWorld |
| **点云 (Point Cloud)** | 3D 点的位置变化 | 适合激光雷达场景 | 稀疏、缺少纹理 | LiDARGen |

## 6. 世界模型 vs 其他 AI 模型

| 特性 | LLM | VLM | VLA | World Model |
| :--- | :--- | :--- | :--- | :--- |
| **输入** | 文字 | 文字+图片 | 文字+图片 | 状态+动作 |
| **输出** | 文字 | 文字 | 动作 | **预测的未来状态** |
| **核心能力** | 语言推理 | 视觉理解 | 物理执行 | **因果预测** |
| **类比** | 大脑的语言区 | 大脑的视觉区 | 运动神经 | **想象力/直觉** |
| **用途** | 聊天、写作 | 看图回答 | 机器人控制 | **规划、仿真** |

## 7. 世界模型为什么现在爆发？

| 驱动力 | 说明 |
| :--- | :--- |
| **视频生成模型成熟** | Sora 等模型证明了大规模视频生成可以涌现物理理解 |
| **VLA 的需求** | 机器人需要"先想后做"——世界模型提供虚拟练习场 |
| **自动驾驶数据需求** | 极端场景（如事故）无法在现实中采集，需要世界模型生成 |
| **算力增长** | GPU 算力足以支撑大规模视频预测和 3D 仿真 |
| **LeCun 的推动** | Yann LeCun 将世界模型定位为通向 AMI（高级机器智能）的核心路径 |

## 8. 世界模型的发展时间线

```
2018    World Models (Ha & Schmidhuber)     ← 奠基论文，VAE+RNN 学习环境模型

2021    Dreamer V2                          ← 离散潜在空间，效率提升
2023.01 Dreamer V3 (Nature)                 ← 🔑 150+ 任务单一配置，Minecraft 钻石
2023.09 GAIA-1 (Wayve)                      ← 自动驾驶世界模型
2023.10 TD-MPC2                             ← 317M 参数跨域世界模型 RL 智能体
2024.02 Genie 1 (DeepMind)                  ← 🔑 首个交互式环境生成
2024.02 Sora (OpenAI)                       ← 🔑 "视频生成模型作为世界模拟器"
2024.02 V-JEPA (Meta)                       ← LeCun 的潜在空间世界模型
2024.05 iVideoGPT                           ← 自回归交互式视频世界模型
2024.12 Genie 2 (DeepMind)                  ← 生成可交互的 3D 世界
2025.01 NVIDIA Cosmos                       ← 🔑 工业级世界基座模型平台
2025.04 GAIA-2 (Wayve)                      ← 多视角可控驾驶世界模型
2025.06 V-JEPA 2 (Meta)                     ← 1.2B 世界模型，零样本机器人规划
2025.08 Genie 3 (DeepMind)                  ← 🔑 实时 24fps/720p 交互世界生成
2026.02 WorldModelBench                     ← 首个物理一致性评测基准
2026.02 World-VLA-Loop                      ← 世界模型+VLA 闭环学习
```

## 9. 世界模型的核心挑战

| 挑战 | 说明 | 现状 |
| :--- | :--- | :--- |
| **长时一致性** | 预测越远，误差累积越大，画面越"飘" | Genie 3 可维持数分钟一致性 |
| **物理真实性** | 物体穿模、违反物理定律 | WorldModelBench 系统评测中 |
| **计算成本** | 视频级别的预测极其昂贵 | 潜在空间预测大幅降低成本 |
| **可交互性** | 能否响应实时动作输入 | Genie 3 实现 24fps 实时交互 |
| **泛化** | 在新场景中预测准不准 | V-JEPA 2 实现零样本迁移 |
| **接触/流体** | 碰撞、水流、布料等复杂物理 | 仍是开放问题 |

## 10. 类比理解

| 类比 | 对应 |
| :--- | :--- |
| 国际象棋选手在脑中推演未来 5 步 | 世界模型预测未来 N 步状态 |
| 建筑师在脑中想象建筑建成后的样子 | 世界模型从当前状态生成目标状态 |
| 司机预判前方车辆会变道 | 自动驾驶世界模型预测交通参与者行为 |
| 厨师想象放入调料后味道的变化 | 机器人世界模型预测操作后物体状态 |

> **一句话总结**：世界模型是 AI 的"想象力引擎"——它让 AI 能在脑中模拟"如果我做了X，世界会变成什么样"，是从反应式智能走向规划式智能的核心技术，被 Yann LeCun 视为通向高级机器智能 (AMI) 的必经之路。
